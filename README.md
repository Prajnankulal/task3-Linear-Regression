# 🏠 House Price Prediction using Linear Regression

## 📌 Project Overview

 The goal is to implement and understand **Simple and Multiple Linear Regression** using a housing dataset to predict house prices.


## 🎯 Objective

* Perform data preprocessing
* Train a Linear Regression model
* Evaluate model performance
* Interpret results


## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn


## 📂 Dataset

Dataset used: **Housing Price Prediction Dataset**
(Source: Kaggle)



## ⚙️ Steps Performed

### 1. Data Loading

* Loaded dataset using Pandas

### 2. Data Preprocessing

* Checked missing values
* Converted categorical variables using one-hot encoding

### 3. Train-Test Split

* Split data into training (80%) and testing (20%)

### 4. Model Training

* Used `LinearRegression` from sklearn

### 5. Prediction

* Predicted house prices on test data

### 6. Model Evaluation

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

### 7. Visualization

* Plotted Actual vs Predicted values

---

## 📊 Results

* The model was able to predict house prices with reasonable accuracy.
* R² score indicates how well the model fits the data.


## 🧠 Key Learnings

* Understanding regression modeling
* Importance of data preprocessing
* Model evaluation techniques
* Interpreting coefficients
